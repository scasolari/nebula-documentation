# Installation

You can install Nebula using yarn, npm and pnpm.

<Tabs items={['yarn', 'npm', 'pnpm']}>
    <Tabs.Tab>
        ```bash
        yarn add @placedv/nebula@0.0.1
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```bash
        npm install @placedv/nebula@0.0.1
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```bash
        pnpm add @placedv/nebula@0.0.1
        ```
    </Tabs.Tab>
</Tabs>

Or import directly in your html

```html
<html>
    <head>
        <script src="https://nebula.placedv.com/nebula/latest/index.js" async type="text/javascript"></script>
    </head>
    <body>
        // ...
    </body>
</html>
```
