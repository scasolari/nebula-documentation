# Installation

To get started with Goofls, you'll need to install the package in your React project. You can do this using either npm, yarn and pnpm.

<Tabs items={['yarn', 'npm', 'pnpm']}>
    <Tabs.Tab>
        ```bash
        yarn add google-fls
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```bash
        npm i google-fls
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```bash
        pnpm add google-fls
        ```
    </Tabs.Tab>
</Tabs>
